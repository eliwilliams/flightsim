This project is a representation of a low-level flight simulator for CS344 (Computer Graphics) at Duke. It is written in Java and uses JOGL, a binding for the OpenGL API.


Controls:
UP/DOWN: Speed up/down
E/S/D/F: Move up/down/left/right
W/R: Pitch up/down
LEFT/RIGHT: Yaw left/right
A/G: Roll left/right
T: Reset scene
C: Change render mode
V: Change shading mode
M: toggle spline curve
N: toggle camera mode
B: toggle control points
X: toggle skybox mode (normal/wireframe/hidden)
Q: Quit (standard with OpenGL)
